# Global-Superstore-Dashboard
 
An interactive Power BI dashboard designed for comprehensive sales and profit analysis of the Global Superstore dataset.

## 🎯 Project Objective
The main goal of this project is to analyze sales data across different regions, categories, and segments to identify key performance drivers, profitable areas, and potential zones for improvement.

## 🔑 Key Insights & Features
* **Sales & Profit Tracking:** Real-time visibility into overall sales and profit margins.
* **Geographical Analysis:** Map visualizations to identify top-performing countries and states.
* **Category Performance:** In-depth breakdown of product categories (Technology, Furniture, Office Supplies) and their profitability.
* **Customer Segmentation:** Analysis of sales behavior across Consumer, Corporate, and Home Office segments.

## 🛠️ Tools Used
* **Power BI Desktop:** For data transformation, modeling, and visualization.
* **Power Query:** For data cleaning and ETL processes.
* **DAX (Data Analysis Expressions):** For creating calculated columns and measures.
